hello hexo

这是我的第一篇博客，介绍如何用hexo框架搭建一个新的blog
1.环境搭建
1.下载node js
nodejs.org
进入上面的网站，下载LTS版本的NODE JS，然后一直确定下一步，安装完成
npm包，先下cnpm包
管理员打开CMD ，先查看NODEjs跟npm 是否下载好了，通过 node -v ;npm -v查看
查看完了之后就是下载一个叫cnpm的东西，因为国内的npm太慢了在CMD里面敲 npm install -g cnpm –registry=https://registry.taobao.org 这个指令来下载cnpm
2.下载hexo框架
在cnpm 下载完了之后 敲 cnpm install -g hexo-cli 来下载这个hexo框架。
2.创建文件夹
1.创建空文件夹
为什么要创建这个文件夹，这个文件夹是你blog的文件夹，如果blog初始化遇到了问题，直接删掉这个文件夹就可以了。
注意这个文件夹是要跟刚刚下载完的这个框架在同一个文件目录下，如果你是在用户目录下下载的框架，那直接在那建文件夹，或者用cmd mkdir命令创建文件夹
2.新建一个HEXO框架
在CMD中进入blog ,也就是刚刚建的目录，然后输入 hexo init 命令来初始化这个Hexo框架，经过等待，初始化完成之后，博客初步搭建完成
可以CMD通过hexo s 来预览你的blog.
